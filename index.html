Колесо фортуны

Строка = сектор на колесе. Картинка — кнопкой у строки; по желанию можно указать URL после текста через Tab.

Варианты

Текст (одна строка — один сектор)

Секторов: 0. Пустые строки — тоже секторы. Ссылку на картинку: текст + Tab + URL (https://…, data:image…) — имеет приоритет над файлом. Цвет сектора — слева от текста.
